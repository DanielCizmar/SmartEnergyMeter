# ESP Energy Consumption Meter

An ESP32-based IoT project for monitoring electrical energy consumption in real time.

This project is designed to measure key electrical parameters such as voltage, current, power, and total energy usage using an external power measurement module. The ESP32 processes the measured data and can later be integrated with monitoring platforms, dashboards, or cloud services for visualization, storage, and analysis.

## Overview

The main goal of this project is to build a compact and practical smart energy meter that can be used for home, laboratory, or IoT monitoring purposes. The system is intended as a foundation for further development, including data logging, remote access, and real-time dashboards.

## Features

- Real-time electrical parameter monitoring
- ESP32-based data processing
- Support for external power measurement modules
- Ready for integration with IoT and cloud platforms
- Suitable for visualization with tools such as InfluxDB and Grafana

## Technologies

- **ESP32**
- **PlatformIO**
- **PZEM / power measurement module**
- **InfluxDB** and **Grafana** (planned integration)
